### 📚 [Security+ SY0-701 - Material de Preparación](./Sec+)
**Sistema completo de estudio** para certificación CompTIA Security+ con sincronización multiplataforma

**Características:**
- ✅ **2292 preguntas** de simulacros de examen (5 dominios completos)
- ✅ **382 flashcards** priorizadas (220 ALTA + 162 MEDIA)
- ✅ **Sistema Round-Robin:** Evita repeticiones hasta agotar todo el banco
- ✅ **5 tipos de pregunta:** Single, Multi-select, Ordering, Matching, True/False
- ✅ **Sincronización PC ↔ Android** vía Syncthing (progreso unificado)
- ✅ **100% offline** con widget Termux (1 tap → listo)
- ✅ **Exportación de errores** a Markdown para repaso dirigido

**Plan de estudio:** 8 semanas estructuradas con laboratorios prácticos
**Objetivo:** Aprobar con 85%+ antes de mayo 2026

**Stack:** HTML5, JavaScript, Python, Termux, Syncthing
